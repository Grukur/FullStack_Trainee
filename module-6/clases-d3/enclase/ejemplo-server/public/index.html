Bienvenido a mi API de Digimones

Instrucciones de uso...

Endpoint GET digimones

Para obtener todos los digimones, deve consultar el siguiente enpoint:

GET: /api/digimones

Endpoint GET digimon por nombre

Para obtener todos los digimones, deve consultar el siguiente enpoint:

GET: /api/digimones/nombre/:nombre

Endpoint GET digimon por level

Para obtener todos los digimones, deve consultar el siguiente enpoint:

GET: /api/digimones/:level
